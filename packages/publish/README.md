# 前端统一研发脚手架

## About

企业内部一站式研发专属脚手架

## Getting Started

安装：

```bash
npm install -g @wqas-cli/core
# OR
yarn global add @wqas-cli/core
```

创建项目

```bash
wqas-cli init 
```

发布项目

```bash
wqas-cli publish
```

## More

DEBUG 模式：

```bash
wqas-cli --debug
```

指定本地包：

```bash
wqas-cli init --packagePath /Users/sam/Desktop/wqas-cli/packages/init/
```
